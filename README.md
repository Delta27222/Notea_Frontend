# Notea_frontend
Frontend for an AI integrated Notes App, developed in TypeScript (NestJs).

## Instalacion de dependecias de la aplicación
```bash
 $ flutter pub get
```
## Comando para correr la aplicación en Flutter
```bash
$ flutter run 
```
